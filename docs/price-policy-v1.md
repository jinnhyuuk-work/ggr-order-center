# 가격 정책 v1

이 문서는 현재 코드 기준 가격 정책의 기준선이다. 리팩터링 중에는 이 문서의 금액이 바뀌지 않아야 하며, 금액 변경은 별도 정책 변경으로 다룬다.

## 1. 공통 원칙

1. 숫자로 계산 가능한 품목은 제품금액에 포함한다.
2. 상담 안내 품목은 제품금액과 예상 결제금액에서 제외한다.
3. 상담 안내 품목은 화면에는 `상담 안내`, payload에는 `null + displayPriceLabel` 형태로 표현한다.
4. 부가세(`vat`)는 현재 모든 페이지에서 `0`으로 계산한다.
5. 최종 예상 결제금액은 `제품금액 + 숫자로 확정된 배송/시공비`이다.

## 2. 합판

합판 자재 단가는 `assets/js/data/board-data.js`에 둔다.

| 카테고리 | 단가 |
| --- | ---: |
| LPM | 47,000원/㎡ |
| PP | 45,000원/㎡ |

계산식:

```text
면적㎡ = (폭mm / 1000) * (길이mm / 1000)
품목금액 = 면적㎡ * ㎡단가 * 수량
```

규격 한계를 넘는 경우 상담 안내로 처리한다.

## 3. 상판

상판 가격 정책은 `assets/js/data/top-data.js`의 `TOP_PRICING_POLICY`에 둔다.

| 카테고리 | 기준 단가 |
| --- | ---: |
| 인조대리석 | 147,000원/m |
| 하이막스 | 210,000원/m |

기준 조건:

| 항목 | 기준 |
| --- | ---: |
| 기준 두께 | 12T |
| 자동 계산 가능 깊이 | 760mm 이하 |
| 금액 올림 단위 | 10원 |
| ㄱ자/역ㄱ자 추가비 | 30,000원 |
| 뒷턱 가용 높이 | 760mm - 상판 깊이 |

과금 길이:

| 형태 | 과금 길이 |
| --- | --- |
| ㅡ자 | `길이` |
| ㄱ자/역ㄱ자 | `길이1 + 길이2 - 깊이` |
| ㄷ자 | `길이1 + 길이2 + 길이3 - 깊이 * 2` |

계산식:

```text
품목금액 = ceil((과금길이mm / 1000) * 카테고리 m단가, 10원)
가공서비스금액 = 형태 추가비 + 선택 가공비
총액 = ceil(품목금액 + 옵션금액 + 가공서비스금액, 10원)
```

12T가 아니거나, 깊이 760mm 초과, 길이 한계 초과, 단가 없는 카테고리는 상담 안내로 처리한다.

## 4. 도어

도어 가격 티어는 `assets/js/data/door-data.js`의 `DOOR_PRICE_TIERS_BY_CATEGORY`에 둔다.

| 카테고리 | 300×800 이하 | 400×800 이하 | 600×800 이하 |
| --- | ---: | ---: | ---: |
| LX SMR PET | 40,000원 | 45,000원 | 50,000원 |
| LX Texture PET | 40,000원 | 45,000원 | 50,000원 |
| Hansol PET | 40,000원 | 45,000원 | 50,000원 |
| LX PET | 35,000원 | 40,000원 | 45,000원 |
| Original PET | 35,000원 | 40,000원 | 45,000원 |
| LPM | 35,000원 | 40,000원 | 45,000원 |

계산식:

```text
품목금액 = 규격 티어 가격 * 수량
경첩가공비 = 유효 경첩 홀 수 * 1,500원 * 수량
총액 = 품목금액 + 옵션금액 + 가공서비스금액 + 경첩가공비
```

600×800mm를 넘는 규격은 상담 안내로 처리한다.

## 5. 시스템

시스템 가격 정책은 `assets/js/data/system-data.js`의 선반/포스트바 티어에 둔다.

일반 선반:

| 폭 | LPM | PP |
| --- | ---: | ---: |
| 400mm 이하 | 22,000원 | 20,000원 |
| 600mm 이하 | 28,000원 | 25,500원 |
| 800mm 이하 | 34,000원 | 31,000원 |

코너 선반:

| 유형 | LPM | PP |
| --- | ---: | ---: |
| 코너 표준 | 42,000원 | 39,000원 |

포스트바:

| 유형 | 2100 이하 | 2300 이하 | 2500 이하 |
| --- | ---: | ---: | ---: |
| 기본 포스트바 | 28,000원 | 28,000원 | 31,000원 |
| 코너 포스트바 | 14,000원 | 14,000원 | 14,000원 |

계산식:

```text
선반금액 = 선반 티어 단가 * 선반 수 * 수량
선반 옵션금액 = 선택 옵션 단가 합계 * 수량
포스트바금액 = 높이 티어 단가 * 개수 * 수량
```

선반 폭 800mm 초과, 선반 길이 한계 초과, 커스텀 가공, 포스트바 높이 2500mm 초과는 상담 안내로 처리한다.
